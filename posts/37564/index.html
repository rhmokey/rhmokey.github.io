层管理(FN按键设置)

Author: Keyboard Date: 2021-05-22  19:50:12

1.层的概念

1.VIA支持多层键值的设定，目前根据型号开放了4层
2.与层概念对应的是键位与键值，肉眼可见的叫键位，每个键位在不同的层都可以设置不同的键值，4个层意味着可以设置4层键值完全不同的布局
3.层可以通过层管理键进行切换
4.层的计数是从0开始，Layer 0 指实际的第一层，Layer 1 指实际的第二层，Layer 2 指实际的第三层，Layer 3 指实际的第四层，

2.实现FN键功能（举例：如果需要实现FN+W等于方向上的功能）

首先在默认 Layer 0 中选择W的键位，并设定键值，然后选择一个键位当FN，通常默认是右Alt旁边的按钮，将键值设定为层管理中的MO(1)，然后在Layer 1的同样键位，将键值设定为UP(方向上)，即可实现上述效果

该过程用到的是层管理中的 MO() 键值

MO() 表示临时切换到某层，按下时键盘会临时切换到指定层，松开时回到当前层

3.使用曾+层管理实现多套配置（举例：需要实现多台电脑甚至多个软件间，使用不同的键值设定）

实现该功能仅需要使用层管理键中的 TO() 键值

TO() 表示切换到指定层，按下后会切换到指定层，并且停留，非主动情况下不会再回来

利用该特性，可以在不同 Layer 中设定一套完全不一样的键值，并在指定键位中放置TO()键值，即可来回切换

MO() 属于“按住”换层，适用于充当FN键功能
TO() 属于“按一下”换层，因为“按一下”后不会切换会之前一层，例如两层切换需要如下设置：在 Layer 0 中设置一个 TO(1) ，之后在 Layer 1 相同位置再设置一个 TO(0) 回到上层
